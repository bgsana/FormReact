# 📋 Formulário de Cadastro com React

Projeto desenvolvido com o objetivo de praticar os fundamentos do React,
com foco em gerenciamento de estado via `useState`, inputs controlados e
componentização de elementos de interface.

## 🚀 Tecnologias

- React
- JavaScript (ES6+)
- CSS

## 📚 Conceitos Aplicados

- **Inputs controlados**: os valores dos campos são gerenciados pelo estado
  do componente, mantendo o React como única fonte da verdade.
- **useState**: hook utilizado para controlar e atualizar os dados do formulário
  em tempo real.
- **Componentização**: os botões foram extraídos em componentes reutilizáveis,
  promovendo organização e reaproveitamento de código.

## 🎯 Objetivo

Exercitar boas práticas no desenvolvimento de formulários com React,
entendendo o fluxo de dados unidirecional e a separação de responsabilidades
entre componentes.
